# httpserver (C#)

 Recreate an http server using TCP sockets. 

 ## Features (WIP)

 Use .Net library to handle incoming HTTP request and TCP connections. 
 Parsed incoming HTTP requests while also contructing the appropriate HTTP response.
 Handle multiple client connections with asyn programming.
Handle routing to different URI paths.

>[!NOTE]
>Does not implement all http features.

## Skill/Tools Used
* .Net
* TCP/IP
* Async programming
* HTTP protocol
* Claude
